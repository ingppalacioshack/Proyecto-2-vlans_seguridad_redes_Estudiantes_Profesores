En la PC ve a "Desktop" > "Web Browser"
En la URL escribe: ftp://IP_SERVIDOR
Usuario Default: cisco / cisco
Si pide autenticación, ingresa las credenciales

File: sampleFile.txt

ftp> dir                 # Listar archivos en el servidor
ftp> get archivo.txt     # Descargar archivo a la PC
ftp> put miarchivo.txt   # Subir archivo al servidor
ftp> delete archivo.txt  # Eliminar archivo del servidor
ftp> mkdir carpeta       # Crear directorio
ftp> cd carpeta          # Cambiar directorio
ftp> pwd                 # Mostrar directorio actual
ftp> quit                # Salir de FTP
....

3. dir
4. get sampleFile.txt
